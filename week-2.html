Workshop Activity:
Paper Prototyping

Week Overview & What We Did



This week, we explored paper prototyping as a rapid web development and UX testing tool. The goal was to create a low-fidelity prototype of a website or simple game
Our process included:
  • Understanding Paper Prototyping – Learning how it helps in UI/UX design.
  • Creating a Paper Prototype – Designing a simple website by assembling paper cut outs.
  • User Testing – Conducting a cognitive walkthrough with peers to refine the design.

Step 1: Measuring the Web

We started with Pixact.ly, an online tool that tests our ability to estimate pixel dimensions. This exercise helped us understand how elements scale on a digital screen, which is useful for designing UI layouts.

Step 2: Creating a Paper Prototype

We were then assigned to sketch a website using paper prototyping techniques.
The prototype needed to represent:

  • Layout structure (navigation, buttons, content sections)
  • User interactions (clicks, scrolls, transitions)
  • Basic functionality (what happens when the user performs an action)

Workshop Activity 2: "Make It Bling"

Week Overview & What We Did



This workshop focused on rapid ideation, wireframing, and styling web pages using CSS. We explored how to create visually engaging designs and improve user experience through consistent styles, colors, and assets.
Our process included:
  • Crazy Eights – Generating 8 quick design ideas in 8 minutes.
  • Wireframing & Sketching – Choosing one idea and refining it.
  • Live Coding Demo – Learning how to use CSS classes, grids, and animations.

Step 1: Crazy Eights

We used the Crazy Eights technique to brainstorm design ideas. In this exercise:

  • We sketched one idea per minute for 8 minutes.
  • The focus was on speed and creativity rather than perfection.
  • At the end, we pinned up and discussed our ideas.

Step 2: Creating a Paper Prototype

We were then assigned to sketch a website using paper prototyping techniques. The prototype needed to represent:

  • Layout structure (navigation, buttons, content sections)
  • User interactions (clicks, scrolls, transitions)
  • Basic functionality (what happens when the user performs an action)

Step 3: Live Coding Demo

We learned how to enhance our designs with CSS styling. Key topics covered:

  • Using CSS Classes & Styles for Consistency
  • Adding Colors, Grids & Assets
  • Extra Styling Features
